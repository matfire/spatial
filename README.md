# Spatial

![GitHub Release](https://img.shields.io/github/v/release/matfire/spatial-tools?style=for-the-badge)


## Install

- install proj
```sh
sudo apt install libproj-dev
```
- install go
- clone this repository
- run `go run main.go` inside the repository's folder
